# Security

`pr-evidence-skill` is local-first and does not call GitHub, publish packages,
or upload logs.

## Reporting

Please report suspected security issues privately to the repository owner before
opening a public issue.

## Handling Evidence

- Do not include secrets, tokens, customer data, or private command output in
  fixtures, examples, issues, or pull requests.
- Prefer short command summaries over full logs.
- Review generated PR bodies before posting them to public repositories.
